# SW-nuclear-fusion
  스마트 시니어 케어 프로토타입

# 순서
  구현할 기능 Issue작성
  Issue번호로 'feature/{Issue번호}'로 new branch 생성 및 작업
  기능 구현이 다 끝났고 오류가 없음을 확인하면 Commit & push
  PR 보낼때 base-develop으로 변경
  코드에 오류 없다고 생각하면 Merge
  checkout 명령어 사용하여 develop으로 이동
  팀원들의 코드가 반영된 develop branch의 코드를 pull
  구현이 끝난 'feature/{Issue번호}' 삭제


프로젝트 제출 시, develop branch에 오류가 없고 정상적으로 코드가 실행된다면
devlop branch를 master branch로 merge
master branch 파일을 제출
